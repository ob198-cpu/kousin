# CDP免許更新管理 独立アプリ

このフォルダは、既存のCDP本体Webアプリとは別URL・別スプレッドシートで運用するための独立Apps Scriptです。

既存URL `https://script.google.com/macros/s/AKfycbwxO4B5BKO1gIsJ2FnXOZwn1bUskpcS3FtdySjsK7A/dev` には push / deploy しないでください。

## 運用ルール

- 免許更新は3年に1度
- 更新講習は有効期限の9か月前から受講可能
- CDPでは有効期限の6か月前から講習・申請受付対象
- 更新講習修了証明は発行から3か月で期限管理
- 有効期限の1か月前から手続き不可ラインとしてP1扱い
- ドローン免許証を確認し、有効期限内かを記録する
- 既存受講生と、新規（他講習機関で受けた人）を顧客区分で分ける

## 初回セットアップ

1. この `renewal_license_app` フォルダを、既存プロジェクトとは別のApps Scriptプロジェクトとして作成します。
2. 作成後、このフォルダ内でだけ `clasp push` / `clasp deploy` を実行します。
3. Webアプリを開き、初回セットアップで専用スプレッドシートを新規作成、または別途用意したスプレッドシートURLを指定します。
4. 設定シートの「内部アラート宛先」に通知先メールアドレスを入れます。
5. 画面の「日次通知設定」で毎日通知を有効にします。

## 推奨機能

- P1/P2だけ日次メール通知
- 個別レコードから内部担当者へ即時メール送信
- 免許証確認日と確認結果を必須管理
- 更新証明期限3か月の自動判定
- 既存/新規（他講習機関）の区分管理
- 次回対応日の手動管理
- アラート送信履歴の自動保存
