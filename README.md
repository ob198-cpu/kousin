# CDP免許更新管理

現在は安全モードです。実データへの接続は停止しています。

## 停止中の機能

- 実データのスプレッドシート読込
- 読み取り元スプレッドシート読込
- 運用ファイル読込
- Googleドキュメント読込
- 対象者情報の保存
- メール送信
- 日次通知トリガー
- 実データIDの保存
- 実URLや実IDの固定記載

## 現在の動作

`Code.js` は空の一覧を返す安全版です。氏名、メール、免許番号、会社名、有効期限などの実データは取得しません。

`Index.html` はサンプル画面を表示します。実データ操作ボタンはありません。

## 再開条件

実データ連携を再開する場合は、事前に以下を明示確認します。

- 接続先スプレッドシートのIDと所有者
- 読み取り範囲
- 書き込み可否
- 画面に表示してよい項目
- メール送信可否
- デプロイ可否

この確認なしに、実データ接続コードを追加しないでください。
